# Migration from Old Dotfiles (Stow)

One-time guide for cleaning up artifacts from the previous GNU Stow-based
dotfiles setup (`~/.dotfiles`). Run on each machine, then delete this file.

## What Changed

| Aspect | Old (stow) | New (chezmoi) |
| --- | --- | --- |
| Manager | GNU Stow (symlinks) | chezmoi (file copies + templates) |
| Repo location | `~/.dotfiles` | `~/.local/share/chezmoi` |
| Prompt | Oh My Posh | Starship |
| Shell config | `.zprofile` + `.config/bash/*.sh` sourced | Single `.zshrc` (templated) |
| Packages | `~/Brewfile` via stow symlink | `~/Brewfile` generated by chezmoi |
| Terminal | iTerm2 | Ghostty |
| Nano | 120+ third-party nanorc files in `~/.nano/` | `scopatz/nanorc` cloned to `~/.nano/`, updated via `dotup` |
| `.hushlogin` | Stow symlink | chezmoi-managed |
| Telemetry | Exports in `.config/bash/exports.sh` | Exports in `.zshrc` |

## Pre-Flight Check

Before cleaning up, verify chezmoi is working on this machine:

```bash
chezmoi doctor        # Should show no errors
chezmoi diff          # Review any pending changes
chezmoi managed       # Confirm files are managed
```

## Step 1: Remove Stow Symlinks

These symlinks point into `~/.dotfiles` and are no longer needed. chezmoi
manages `.zshrc` directly; the rest are not carried forward.

```bash
# Remove stow symlinks (safe — targets still exist in ~/.dotfiles)
rm -f ~/.zprofile
rm -f ~/.bash_profile
rm -f ~/.bashrc
rm -f ~/.hushlogin    # chezmoi will recreate this as a regular file

# Remove stow symlink directories
rm -f ~/.config/bash    # symlink to ../.dotfiles/bash/.config/bash
rm -f ~/.config/iterm   # symlink to ../.dotfiles/iterm/.config/iterm
```

> **Note**: `~/.zshrc` is managed by chezmoi (not a symlink). Do NOT remove it.
> `~/Brewfile` is also chezmoi-managed now. Do NOT remove it.

## Step 2: Remove Oh My Posh Artifacts

The new setup uses Starship instead of Oh My Posh.

```bash
# macOS
brew uninstall oh-my-posh 2>/dev/null

# Linux (if installed to ~/bin)
rm -f ~/bin/oh-my-posh

# All platforms
rm -rf ~/.cache/ohmyposh
```

## Step 3: Replace Old Nano Syntax Files

The old setup installed nanorc files from a third-party repo into `~/.nano/`
as loose files. The new setup clones `scopatz/nanorc` as a git repo to the
same location, and updates it via `dotup`.

```bash
# Remove old (non-git) nano files — chezmoi will re-clone as a git repo
rm -rf ~/.nano
```

After running `chezmoi apply` (or the install script), `~/.nano` will be
recreated as a git clone of `scopatz/nanorc`.

## Step 4: Remove Stow (if No Longer Needed)

```bash
# macOS
brew uninstall stow

# Ubuntu/Debian
sudo apt-get remove -y stow
```

## Step 5: Remove the Old Repo

Once all machines are migrated and you're confident everything works:

```bash
rm -rf ~/.dotfiles
```

> **Safety**: The archive is also at `~/dev/dotfiles-archive` and on GitHub.
> Nothing is lost by removing `~/.dotfiles`.

## Step 6: Verify

```bash
# No stale symlinks in home
find ~ -maxdepth 1 -type l -ls 2>/dev/null

# No stale symlinks in .config
find ~/.config -maxdepth 1 -type l -ls 2>/dev/null

# chezmoi is healthy
chezmoi doctor
chezmoi diff
```

## Platform-Specific Notes

### macOS

- iTerm2 preferences (`~/.config/iterm/com.googlecode.iterm2.plist`) were
  loaded via `defaults write`. Removing the symlink does not undo the imported
  settings. If you've switched to Ghostty, no action needed.
- The old Brewfile had packages not in the new setup (AWS CLI, Azure CLI,
  Kubernetes tools, Java/Gradle, etc.). These remain installed via Homebrew
  but are no longer managed. Run `brew leaves` to audit.

### Ubuntu / WSL

- The old setup installed PowerShell Core via Microsoft's APT repo. If no
  longer needed: `sudo apt-get remove -y powershell`
- `~/bin/` may contain the old Oh My Posh binary. Remove with Step 2.
- `~/dev/personal`, `~/dev/work`, `~/dev/scratch` directories were created by
  the old installer. Keep or remove as you prefer.

### Windows

- The old setup used `win/win-term-profile.ps1` and `win/python-setup.ps1`.
  These were run manually, not stowed. No cleanup needed unless you want to
  review Windows Terminal settings.

## Full Cleanup Script

Copy-paste for a single machine (macOS). Review before running:

```bash
#!/bin/bash
set -euo pipefail

echo "=== Removing stow symlinks ==="
rm -fv ~/.zprofile ~/.bash_profile ~/.bashrc ~/.hushlogin
rm -fv ~/.config/bash ~/.config/iterm

echo "=== Removing Oh My Posh ==="
brew uninstall oh-my-posh 2>/dev/null || true
rm -rf ~/.cache/ohmyposh

echo "=== Removing nano syntax files ==="
rm -rf ~/.nano

echo "=== Removing stow ==="
brew uninstall stow 2>/dev/null || true

echo "=== Removing old dotfiles repo ==="
rm -rf ~/.dotfiles

echo "=== Applying chezmoi (recreates .hushlogin, .nanorc, etc.) ==="
chezmoi apply -v

echo "=== Verifying ==="
echo "Stale symlinks in ~:"
find ~ -maxdepth 1 -type l -ls 2>/dev/null || echo "  (none)"
echo "Stale symlinks in ~/.config:"
find ~/.config -maxdepth 1 -type l -ls 2>/dev/null || echo "  (none)"
chezmoi doctor
echo "=== Done ==="
```
